# seed-data (optionnel)

Les leçons Ecodim n’ont **plus besoin** d’un PDF placé ici : le bouton « Télécharger le PDF » génère le document à partir du contenu texte en base (`/api/courses/[id]/pdf`).

Ce dossier reste utile si vous souhaitez conserver d’autres fichiers locaux pour des scripts ou imports manuels via l’administration.
